# Customer Orders Analysis Using Python

## Overview
This project analyzes customer orders using Python data structures such as lists, tuples, dictionaries, and sets.

## Features
- Store customer orders
- Classify products by category
- Calculate customer spending
- Identify high-value customers
- Generate business insights using loops and conditionals

## Technologies Used
- Python
- Lists
- Tuples
- Dictionaries
- Sets
- Loops and Conditionals

## How to Run
1. Download the project
2. Open in VS Code
3. Run:
python customer_orders.py
